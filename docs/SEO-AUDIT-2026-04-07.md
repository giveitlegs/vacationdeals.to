## Critical Issues (Must Fix)

### C1. Missing Canonical URLs on Some Pages
**Pages affected:** /destinations, /brands
**Impact:** HIGH ‚Äî Could cause duplicate content indexing
**Fix:** Add `<link rel="canonical" href="...">` to all pages. Check `generateMetadata()` in destinations/page.tsx and brands/page.tsx.

### C2. Some Brands Show "0 Deals Available"
**Pages affected:** /brands page shows some brands with 0 deals
**Impact:** HIGH ‚Äî Thin content signals to Google, poor user experience
**Fix:** Either hide brands with 0 active deals from the listing, or ensure all scrapers are producing deals. Check: GoVIP, Monster Vacations (parked domains), and any brands with broken scrapers.

---

## High Priority Issues

### H1. Images Missing Alt Text
**Pages affected:** Site-wide
**Impact:** MEDIUM-HIGH ‚Äî Accessibility failure, missed image SEO opportunities
**Details:** Deal cards use gradient `div` backgrounds with `role="img"` and `aria-label`, but:
- SVG decorative icons lack alt descriptions
- Some resort card images have minimal alt text
- City destination cards use CSS backgrounds instead of `<img>` tags
**Fix:** Add descriptive alt text to all images. For CSS background images, ensure `aria-label` is descriptive (e.g., "Westgate Las Vegas Resort pool view" not just "deal card").

### H2. Duplicate Deal Ticker Rendering
**Pages affected:** Multiple pages (blog posts, destination pages)
**Impact:** MEDIUM ‚Äî The VacPack Rate Ticker appears twice identically at page top
**Fix:** Ensure the ticker component renders only once per page in the layout.

### H3. OG Image Missing on Blog Index
**Pages affected:** /blog
**Impact:** MEDIUM ‚Äî Social sharing will use a generic fallback image
**Fix:** Set a dedicated OG image for /blog (e.g., a branded blog header graphic).

### H4. Destination Pages with Incomplete Location Data
**Pages affected:** /destinations listing
**Impact:** MEDIUM ‚Äî Some destinations show trailing commas suggesting missing state/country data
**Fix:** Clean up destination seed data in `packages/db/src/seed.ts` ‚Äî ensure all 97 destinations have complete city, state, country fields.

## Medium Priority Issues

### M1. Title Tags Exceeding 60 Characters
**Pages affected:** Some deal detail pages, blog posts
**Example:** "Travel Deal Tuesday - View Exclusive Offer ‚Äî 3-Night Stay from $59 | VacationDeals.to" (85 chars)
**Fix:** Truncate deal titles in `generateMetadata()` to keep total under 60 chars. Consider format: "[Resort] [City] [Duration] from $[Price]"

### M2. Blog Post Templates ‚Äî Thin Content Risk
**Pages affected:** Some brand/destination blog posts
**Impact:** MEDIUM ‚Äî Deal descriptions within blog posts can be brief and templated
**Fix:** Enrich blog posts with unique resort descriptions, local tips, and differentiated content.

### M3. Deal Card H3 Tags Are Generic
**Pages affected:** /deals, destination pages, brand pages
**Impact:** LOW-MEDIUM ‚Äî Some deal H3s like "Travel Deal Tuesday - View Exclusive Offer" are vague
**Fix:** Use more descriptive headings: "3-Night Orlando Resort Stay from $59 at Westgate"

### M4. Missing `lastmod` on Blog Articles in Schema
**Pages affected:** /blog listing
**Impact:** LOW-MEDIUM ‚Äî Google can't determine freshness from schema alone
**Fix:** Add `dateModified` to Article schema and individual blog post metadata.

### M5. Some Deals Missing Complete Information
**Pages affected:** GetawayDealz deal cards
**Impact:** LOW-MEDIUM ‚Äî Cards lacking descriptions appear thin
**Fix:** Ensure all scrapers extract description/inclusions fields.
