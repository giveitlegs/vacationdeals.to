# El Cid Vacations Club — banner mockup spec

**Slug:** `el-cid`

**Tone:** Mexico-focused, all-inclusive heritage brand

**Primary color:** #1B3F6B (royal blue)

**Secondary color:** #D4AF37 (gold)

**Signature callout:** Mazatlan + Riviera Maya all-inclusive


## How to use these prompts

1. Open Nano Banana Pro (or your image gen of choice)

2. Paste each prompt below

3. Save the output as `apps/web/public/banners/el-cid-<W>x<H>.png`

4. After all four are generated, run `scripts/seed-prospect-banners.sql` to wire them up

5. Send the prospect: `https://vacationdeals.to/?utm_content=el-cid-prospect` — they see only their banner at the top of every page

## leaderboard — 970×90 (Header banner — wide)

```text
Create a 970x90 pixel display ad banner for "El Cid Vacations Club" advertising vacation packages on VacationDeals.to.

REQUIRED ELEMENTS:
- Brand logo treatment "El Cid Vacations Club" prominently on the LEFT (or top if vertical), in a clean modern sans-serif (DO NOT copy actual trademarked logo — original wordmark in brand palette)
- Headline: "Mazatlan + Riviera Maya all-inclusive"
- Sub-headline: "Book Your Stay Today"
- CTA button: "Book Now" or "View Deal"
- Tasteful "vacationdeals.to" attribution at bottom-right in small text
- Background: Mazatlan + Riviera Maya beach scenes with Spanish colonial accents

VISUAL STYLE:
- Tone: Mexico-focused, all-inclusive heritage brand
- Primary color: #1B3F6B (royal blue)
- Secondary/accent color: #D4AF37 (gold)
- High-quality marketing photography aesthetic; no stock-photo cliches
- Composition optimized for 970x90 (wide horizontal)
- Text must be legible against background — use overlay tints if needed
- No watermarks, no Shutterstock/Getty marks
- Photorealistic, premium feel — NOT cartoon or AI-obvious

NEGATIVE: no text errors, no garbled words, no extra random logos, no trademarked third-party logos, no people staring directly at camera, no slogans not listed above.

Output: a single 970x90 PNG, transparent background OK if it improves placement contrast.
```
