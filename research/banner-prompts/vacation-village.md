# Vacation Village Resorts — banner mockup spec

**Slug:** `vacation-village`

**Tone:** Direct resort brand, value-tier

**Primary color:** #005EB8 (resort blue)

**Secondary color:** #FBC91B (yellow)

**Signature callout:** From $49 — direct resort pricing


## How to use these prompts

1. Open Nano Banana Pro (or your image gen of choice)

2. Paste each prompt below

3. Save the output as `apps/web/public/banners/vacation-village-<W>x<H>.png`

4. After all four are generated, run `scripts/seed-prospect-banners.sql` to wire them up

5. Send the prospect: `https://vacationdeals.to/?utm_content=vacation-village-prospect` — they see only their banner at the top of every page

## leaderboard — 970×90 (Header banner — wide)

```text
Create a 970x90 pixel display ad banner for "Vacation Village Resorts" advertising vacation packages on VacationDeals.to.

REQUIRED ELEMENTS:
- Brand logo treatment "Vacation Village Resorts" prominently on the LEFT (or top if vertical), in a clean modern sans-serif (DO NOT copy actual trademarked logo — original wordmark in brand palette)
- Headline: "From $49 — direct resort pricing"
- Sub-headline: "Book Your Stay Today"
- CTA button: "Book Now" or "View Deal"
- Tasteful "vacationdeals.to" attribution at bottom-right in small text
- Background: Mid-tier Florida or Williamsburg resort, pool + kids, bright daylight

VISUAL STYLE:
- Tone: Direct resort brand, value-tier
- Primary color: #005EB8 (resort blue)
- Secondary/accent color: #FBC91B (yellow)
- High-quality marketing photography aesthetic; no stock-photo cliches
- Composition optimized for 970x90 (wide horizontal)
- Text must be legible against background — use overlay tints if needed
- No watermarks, no Shutterstock/Getty marks
- Photorealistic, premium feel — NOT cartoon or AI-obvious

NEGATIVE: no text errors, no garbled words, no extra random logos, no trademarked third-party logos, no people staring directly at camera, no slogans not listed above.

Output: a single 970x90 PNG, transparent background OK if it improves placement contrast.
```
