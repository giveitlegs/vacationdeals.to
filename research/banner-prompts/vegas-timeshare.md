# Las Vegas Timeshare — banner mockup spec

**Slug:** `vegas-timeshare`

**Tone:** Vegas-focused, casino-adjacent

**Primary color:** #000000 (black)

**Secondary color:** #FFD700 (gold)

**Signature callout:** SAHARA + Planet Hollywood + Westgate Vegas


## How to use these prompts

1. Open Nano Banana Pro (or your image gen of choice)

2. Paste each prompt below

3. Save the output as `apps/web/public/banners/vegas-timeshare-<W>x<H>.png`

4. After all four are generated, run `scripts/seed-prospect-banners.sql` to wire them up

5. Send the prospect: `https://vacationdeals.to/?utm_content=vegas-timeshare-prospect` — they see only their banner at the top of every page

## leaderboard — 970×90 (Header banner — wide)

```text
Create a 970x90 pixel display ad banner for "Las Vegas Timeshare" advertising vacation packages on VacationDeals.to.

REQUIRED ELEMENTS:
- Brand logo treatment "Las Vegas Timeshare" prominently on the LEFT (or top if vertical), in a clean modern sans-serif (DO NOT copy actual trademarked logo — original wordmark in brand palette)
- Headline: "SAHARA + Planet Hollywood + Westgate Vegas"
- Sub-headline: "Book Your Stay Today"
- CTA button: "Book Now" or "View Deal"
- Tasteful "vacationdeals.to" attribution at bottom-right in small text
- Background: Vegas Strip skyline at night, neon lights, casino-luxe

VISUAL STYLE:
- Tone: Vegas-focused, casino-adjacent
- Primary color: #000000 (black)
- Secondary/accent color: #FFD700 (gold)
- High-quality marketing photography aesthetic; no stock-photo cliches
- Composition optimized for 970x90 (wide horizontal)
- Text must be legible against background — use overlay tints if needed
- No watermarks, no Shutterstock/Getty marks
- Photorealistic, premium feel — NOT cartoon or AI-obvious

NEGATIVE: no text errors, no garbled words, no extra random logos, no trademarked third-party logos, no people staring directly at camera, no slogans not listed above.

Output: a single 970x90 PNG, transparent background OK if it improves placement contrast.
```
