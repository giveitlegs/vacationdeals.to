# Pueblo Bonito Resorts — banner mockup spec

**Slug:** `pueblo-bonito`

**Tone:** Premium Mexico destination

**Primary color:** #8B0000 (deep terracotta)

**Secondary color:** #F4D03F (sunny gold)

**Signature callout:** Cabo + Mazatlan luxury


## How to use these prompts

1. Open Nano Banana Pro (or your image gen of choice)

2. Paste each prompt below

3. Save the output as `apps/web/public/banners/pueblo-bonito-<W>x<H>.png`

4. After all four are generated, run `scripts/seed-prospect-banners.sql` to wire them up

5. Send the prospect: `https://vacationdeals.to/?utm_content=pueblo-bonito-prospect` — they see only their banner at the top of every page

## leaderboard — 970×90 (Header banner — wide)

```text
Create a 970x90 pixel display ad banner for "Pueblo Bonito Resorts" advertising vacation packages on VacationDeals.to.

REQUIRED ELEMENTS:
- Brand logo treatment "Pueblo Bonito Resorts" prominently on the LEFT (or top if vertical), in a clean modern sans-serif (DO NOT copy actual trademarked logo — original wordmark in brand palette)
- Headline: "Cabo + Mazatlan luxury"
- Sub-headline: "Book Your Stay Today"
- CTA button: "Book Now" or "View Deal"
- Tasteful "vacationdeals.to" attribution at bottom-right in small text
- Background: Cabo San Lucas + Mazatlan resort architecture, Pacific sunset

VISUAL STYLE:
- Tone: Premium Mexico destination
- Primary color: #8B0000 (deep terracotta)
- Secondary/accent color: #F4D03F (sunny gold)
- High-quality marketing photography aesthetic; no stock-photo cliches
- Composition optimized for 970x90 (wide horizontal)
- Text must be legible against background — use overlay tints if needed
- No watermarks, no Shutterstock/Getty marks
- Photorealistic, premium feel — NOT cartoon or AI-obvious

NEGATIVE: no text errors, no garbled words, no extra random logos, no trademarked third-party logos, no people staring directly at camera, no slogans not listed above.

Output: a single 970x90 PNG, transparent background OK if it improves placement contrast.
```
